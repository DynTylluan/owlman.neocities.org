Getting around Neocities' restrictions.

If you know Neocities quite well, you're fully aware of whats happening. If you don't, let me explain.

So Neocities has some file restrictions on what kinds of files you're able to put on your website, which sucks because you cant add music or videos to your website. The only kind of music you can add is simple MIDI files, but I found a way to get over that whitelist.

There's this website that you can go to that can convert any file into Base64. Basically, FILE to TEXT. So you can drag a video or music file and turn it into text, which then you can use instead of a path to a file.

You can find it here. So just throw in a file, copy and paste the file contents into the src and then BAM! You got around the file restrictions.

You could just upgrade to supporter, but that costs money. If you're like me, you don't have any.
